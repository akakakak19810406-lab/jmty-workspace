# skills.sh 導入・活用マニュアル

skills.sh は、AIエージェント向けの再利用可能な「スキル（機能）」を簡単に導入・管理できるコマンドラインツールです。
このツールを使うことで、世界中の開発者が公開している便利なスキルを、あなたのエージェントに素早く追加することができます。

ウェブサイト: [skills.sh](https://skills.sh/)

## 1. 導入方法 (インストール)

skills.sh は `npx` コマンドを使って実行するため、事前のインストール作業は不要です。
ターミナルで以下のコマンドを実行するだけで、すぐに使い始めることができます。

```bash
npx skills --help
```

毎回 `npx` を打つのが手間な場合は、以下のコマンドでグローバルにインストールすることも可能です（必須ではありません）。

```bash
npm install -g skills
# インストール後は `skills` コマンドだけで実行できます
# 例: skills list
```

## 2. 基本的な使い方

### 2.1 新しいスキルを追加する

スキルを追加するには `add` コマンドを使用します。
追加したいスキルは [skills.sh](https://skills.sh/) の公式サイトで探すことができます。

**構文:**
```bash
npx skills add <ユーザー名/リポジトリ名>
```

**例:**
Vercel Labs が提供するエージェントスキル集を追加する場合：
```bash
npx skills add vercel-labs/agent-skills
```

※ コマンドを実行すると、プロジェクト内の `.agent/skills` フォルダにスキルがダウンロードされます。

### 2.2 インストール済みスキルの一覧を確認する

現在導入されているスキルを確認するには `list` コマンドを使います。

```bash
npx skills list
```

### 2.3 スキルを探す

コマンドラインから対話的にスキルを検索することもできます。

```bash
npx skills find
```
またはキーワードを指定して検索：
```bash
npx skills find <キーワード>
```

### 2.4 スキルを更新する

導入済みのスキルを最新版に更新します。

```bash
npx skills update
```

### 2.5 スキルを削除する

不要になったスキルを削除します。

```bash
npx skills remove
```
（削除したいスキルをリストから選択できます）

---

## 3. 補足情報

- 導入されたスキルは、現在のプロジェクトの `.agent/skills/` ディレクトリに保存されます。
- このリポジトリでは、スキルの正本は `.agent/skills/` のみです。
- 導入後は `AGENTS.md` のスキル一覧も最新にそろえる前提で運用します。
- 各スキルの詳細な使い方は、`.agent/skills/<スキル名>/SKILL.md` ファイルを参照してください。
- 既存のスキル（`git-workflow` や `jmty-posts` など）もこのコマンドで一覧表示・管理が可能です。
