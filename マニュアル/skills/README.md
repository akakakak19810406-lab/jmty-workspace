# スキルマニュアルショートカット

このフォルダは `.agent/skills/` 内の各 `SKILL.md` へのシンボリックリンク集です。
直接開いてスキル定義を参照できます。

## 命名ルール
- `*_skill.md` → `SKILL.md` へのシンボリックリンク
- `*_manual.md` → `MANUAL.md` へのシンボリックリンク

## 追加ルール
- 新規スキル作成時は、このフォルダにシンボリックリンクを追加し、この README も更新する。
- リンク先: `../../.agent/skills/{カテゴリ}/{スキル名}/SKILL.md`
- `AGENTS.md` の `### Available skills` も同時に更新すること。
