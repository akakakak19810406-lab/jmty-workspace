# スラッシュコマンド一覧

この一覧は JMTY workspace で使う repo 独自コマンドの早見表です。
Claude Code 標準コマンドとは別物として扱います。

## モード切り替え

| コマンド | 何をするか |
|---|---|
| `/team` | チーム共有の作業モードにする |
| `/personal` | 個人作業モードにする。成果物は `personal/<account>/` 配下へ置く |

## Git・運用

| コマンド | 何をするか |
|---|---|
| `/setup` | JMTY workspace の初期状態を確認する |
| `/pull` | `origin/main` の最新変更を取り込む |
| `/c` | 今の変更をコミットする |
| `/git` | 変更をリモートへ反映し、必要なら Discord 報告する |
| `/git-nd` | 変更をリモートへ反映する。Discord 報告はしない |

## JMTY 制作

| コマンド | 何をするか |
|---|---|
| `/jmty` | 工場求人または在宅求人の投稿文を作る |
| `/jmty-weekly` | ジモティ週次素材処理を起動する |
| `/ceo` | 依頼内容を見て適切なスキルへ振り分ける |

## Claude Code 標準コマンド

`/loop`、`/schedule`、`/btw`、`/branch`、`/remote-control`、`/mobile`、`/hooks`、`/voice` は Claude Code 本体の機能です。
