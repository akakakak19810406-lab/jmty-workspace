# スラッシュコマンド一覧

チャットに `/コマンド名` と入力するだけで、AI が動いてくれます。

> Claude Code には、team-info 独自コマンドとは別に標準コマンドもあります。`/loop`、`/schedule`、`/btw`、`/branch`、`/remote-control`、`/mobile`、`/hooks`、`/voice` は Claude Code 本体の機能です。

---

## モード切り替え（最初に設定する）

作業が「チームに共有するもの」か「自分だけのもの」かを決めます。

| コマンド | 何をするか |
|---------|-----------|
| `/team` | チームモードにする。作ったファイルはチームと共有される |
| `/personal` | 個人モードにする。作ったファイルは自分だけに残り、チームには見えない |

> 今どっちのモードか分からなくなったら `/team` か `/personal` を入力するだけでOK。

## 開発・制作のコマンド

| コマンド | 何をするか |
|---------|-----------|
| `/pull` | チームの最新の変更を自分のパソコンに取り込む（作業前に必ず使う） |
| `/c` | 今の変更を保存する（チームには送らない。こまめに使う） |
| `/git` | 変更をチームに送る。終わったあと Discord 報告をするか聞く |
| `/git-nd` | 変更をチームに送る。Discord 報告はしない |
| `/script` | 動画の台本を作る |
| `/voice` | legacy 名。Claude Code では標準の音声入力とぶつかるため、VOICEVOX 音声化は「`voice-script-launcher` を使って」と頼む |
| `/lyric` | 歌詞動画を作る |
| `/sleep-travel` | 「寝ながらトラベル」の動画を作る |
| `/acoriel` | 「アコリエル」のアコースティックカバー動画を作る |
| `/jmty` | ジモティーの投稿文を作る |

## Claude Code 標準の便利機能

| コマンド | 何をするか |
|---------|-----------|
| `/loop` | 今の session の中で、一定間隔の見守りや確認を回す |
| `/schedule` | Claude Code の標準スケジューラで定期実行や一回だけのリマインドを作る |
| `/btw` | 本筋を汚さずに横道の質問をする |
| `/branch` | 今の会話を途中から分岐する |
| `/remote-control` | スマホやブラウザから、今このパソコンで動いている session を続ける |
| `/mobile` | Claude のスマホアプリ導線を開く |
| `/hooks` | Claude Code に入っている hook 設定を見る |

> 定期実行はまず「読むだけ・確認するだけ」の用途から始めます。ファイル編集、Git、Docker、サーバー操作を自動化したいときは、先に確認を取ります。
