# ジモティー画像プロンプトテンプレート

このフォルダに画像生成用テンプレートを保存します。

## ファイル名ルール
- 工場求人だけに使う: `factory_*.md` または `工場_*.md`
- 在宅求人だけに使う: `remote_*.md` または `在宅_*.md`
- 全種別で使う: `common_*.md` または `共通_*.md`

## 使える差し込み項目
- `{{account_name}}`: アカウント名
- `{{task_type}}`: `factory` / `remote1` / `remote2`
- `{{region}}`: 投稿先地域
- `{{salary_text}}`: 給与表記
- `{{role_phrase}}`: 職種表記
- `{{post_text}}`: 投稿文章

週次処理では、対象に合うテンプレートからアカウント・行番号・種別ごとに安定したランダム選択をします。
