# inputs/ — JMTY 素材インボックス

ここはジモティ投稿を作るための入力素材置き場です。
生成済み投稿文や整理後の成果物は `outputs/` へ置き、このフォルダには保存しません。

## サブフォルダ構造

| フォルダ | 用途 | 対応スキル |
|---|---|---|
| `jmty_factory_cases/` | 工場求人の案件素材 | `jmty-posts` |
| `jmty_remote_samples/` | 在宅求人の見本・職種リスト | `jmty-posts` |

## 注意

- 入力素材は原則 Markdown / テキストで管理します。
- 個人作業用の素材は `personal/<account>/inputs/` へ置きます。
- フォルダ名は既存の `jmty_*` 形式に合わせます。
